# Lazyfair Vue3 + TypeScript

Lazyfair public website 

## Project Setup

```sh
npm install
```

### Compile and Hot-Reload for Development

```sh
npm run dev
```

### Type-Check, Compile and Minify for Production

```sh
npm run build
```

### SSH to cloud server

```sh
sudo ssh -i lazyfair-vue.pem ubuntu@52.65.228.90
```
